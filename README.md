# Mobile Debug Environment (SESL v.0.3)

## 🎯 О проекте
Это профессиональная среда для эмуляции мобильных устройств (390px/768px) прямо внутри VS Code. Создана для тех, кто хочет видеть «идеальный» результат верстки без тяжелых внешних симуляторов.

Эмулятор работает как **«контрольное лекало» (Reference Frame)**:
* Он **не** исправляет неадаптивный сайт автоматически.
* Он создает жесткие границы (390px/768px), внутри которых вы обязаны подогнать медиа-стили.

**Цель:** Добиться 100% совпадения «идеального вида» в эмуляторе и «реального вида» в мобильном браузере. Если в эмуляторе всё красиво, а в браузере криво — значит, медиа-запросы в CSS написаны неверно.

## 🤖 Hybrid Human-AI Collaboration
Проект настроен для совместной работы человека и ИИ-агентов. Все правила взаимодействия, включая распределение долей вклада (33% Основателю / 67% Коллаборантам), описаны в файле [CONTRIBUTING.md](./CONTRIBUTING.md).

## 🚀 Как использовать
1. Склонируйте этот репозиторий в папку со своим веб-проектом.
2. Подробная инструкция по запуску эмулятора находится здесь: [README.md](../README.md).
